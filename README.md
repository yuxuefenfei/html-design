# 送给我亲爱的宝贝-孟颖

# [姓名]的生日惊喜网页项目 (Birthday Surprise for Meng Ying)

本项目是一套以“千禧龙年（2000年）”为主题，专为[姓名]打造的极致浪漫惊喜网页集。项目采用轻量且优雅的纯 HTML/CSS/JS 编写，无任何外置依赖，支持任意现代设备流畅运行全尺寸互动动画。

为了便于个性化修改，所有的页面动画引擎均与内容实现了深度解耦。你只需要通过任何文本编辑器打开对于文件，修改其 HTML 内部最上方 `<script>` 标签内的 `CONFIG` 变量即可完美定制专属于你们的浪漫内容。

---

## 页面功能总览与参数配置指南

### 1. 星河溯流 (`idea1_stars.html`)
- **动效功能**：满屏如呼吸般温柔的梦幻星野，当点击屏幕中央后，所有散落漂浮的星辰将极具科幻感地收拢、加速溯流，并且最终不可思议地汇聚成受赠者的名字。
- **参数配置 (`CONFIG`)**：
  - `targetName`: 星光最终汇聚形成的目标文字（目前默认：`'[姓名]'`）。
  - `modalTitle`: 右侧寓意讲解的标题（默认：`'星光汇聚'`）。
  - `modalDesc`: 寓意面板内部详细文案。

### 2. 解锁你的心 (`idea2_password.html`)
- **动效功能**：沉浸式的暗场密码验证系统。输入设定好的特定密码或生日口令后，会瞬间解开冰封，深红色的全屏心跳和四溅的礼花在屏幕绽放，同时从阴影中浮现出终极浪漫告白。
- **参数配置 (`CONFIG`)**：
  - `passwords`: 放行通关的密码库白名单数组（例如：`['[姓名]', '2000', '[pinyin]', '[单字]']`，只要输入的值能匹配这个数组的任一项就能触发彩蛋）。
  - `passHint`: 输入框上方的预提示文字（例：`'通关密码是：属于我的小龙女'`）。
  - `welcomeMessage`: 解锁成功后，全屏展示的巨幕欢迎字幕。
  - `modalTitle` & `modalDesc`: 右侧寓意弹窗相关的文本配置。

### 3. 专属时空轴 (`idea3_timeline.html`)
- **动效功能**：无限向下的纵向沉浸时间轴。使用 GPU `will-change` 以及动态观察器呈现，在滑动浏览岁月刻度时，代表着特定记忆的文字卡片会一页一页带着光晕从暗处柔和显影。
- **参数配置 (`CONFIG`)**：
  - `heroTitle`: 页面初始化时的中心巨浪名片称谓（例：`'2000 · 龙之女'`）。
  - `events`: **核心骨架配置（无限数组）**。该对象内部包含所有你想要讲述的故事：
    只需像这样往下写死 `{ year: '年份/时间', desc: '详细描述' }`，网页会自动在视觉中为你接起一条无限长的排版完美的时间主线！
  - `modalTitle` & `modalDesc`: 寓意相关文案配置。

### 4. 火漆实体信 (`idea4_letter.html`)
- **动效功能**：还原真实物理厚度的 3D 专属信件动画。当点击带有真实阴影的复古火漆印盖时，信封盖将会随着平滑曲线打开，紧接着信纸本身会自动抽出，错落有致地浮现出里面蕴藏的寄语段落。
- **参数配置 (`CONFIG`)**：
  - `letterTitle`: 取出信件时，抬头的第一行高亮文案（例：`'致 我的[姓名]'`）。
  - `letterParagraphs`: **信封深意内容（数组）**。数组里存着的每一句话，就会被视为信封段落的一行。你可以无限追加想要对她说的话，系统会自动为这行新的话套上“依次滑动、缓出延迟”的高级呈现动效，杜绝生硬感！
  - `modalTitle` & `modalDesc`: 寓意相关文案配置。

### 5. 同频共振的心 (`idea5_heart.html`)
- **动效功能**：极具冲击力的纯黑底色浪漫爱心，自带律动的呼吸光晕，点击爱心即可爆发炫丽动感的动态爆破粒子碎片。
- **页面寓意**：“哪怕世界再嘈杂，只要想起你，我的心跳就会同频共振。”
- **无感配置**：此页为纯动画交互引擎母版，点击右上方“寓意”按钮即可查看寄语。
